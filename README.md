## Description

Todo app server, featuring NestJS.

## Project setup

```bash
$ git clone https://github.com/iliasse-e/Todo-app-server.git
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
